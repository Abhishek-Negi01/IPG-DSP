# Frontend

React web interface for citizens and government officials.

## What It Does

- Citizens can submit grievances through a form
- Government officials can view processed grievances and statistics
- Shows AI processing results (category, urgency, department routing)

## Pages

- `/` - Home page
- `/submit` - Grievance submission form
- `/dashboard` - Government analytics
- `/grievances` - List of all grievances

## How to Run

```bash
cd frontend
npm install
npm run dev
```

Web app runs at http://localhost:3000

Built with React + Tailwind CSS.
